- TypeScript
- Tailwind CSS
- Next/Image
- REST API Integration

## Getting Started

### Prerequisites

- Node.js 18.x or later
- npm

### Installation

1. Clone the repository:

```
git clone https://github.com/nuruzzamn/product-list.git
```

2. Go to project directory:

```
cd product-list
```

3. Installing dependancies:

```
npm install
```

4. Run the the project:

```
npm run dev
```
